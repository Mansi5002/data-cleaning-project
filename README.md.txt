# Data Cleaning Project

This project focuses on basic data validation and cleaning using Python and pandas.

I worked with a dataset containing 1000 records of people information such as name, email, gender, date of birth, phone number, and job title.

## What I Did

- Loaded the dataset using pandas
- Checked for missing values
- Checked for duplicate records
- Removed unnecessary columns
- Converted the date of birth column to datetime format
- Cleaned phone numbers by removing special characters
- Standardized text fields (lowercase and trimmed spaces)
- Saved the cleaned dataset as a new CSV file

## Tools Used

- Python
- Pandas
- Jupyter Notebook

The final cleaned dataset is saved as:
cleaned_people_1000.csv
